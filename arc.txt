VS Code Ecosystem:
│
├── EDITING ENVIRONMENT (settings.json)
│   ├── Needs: Python path for IntelliSense  
│   ├── Timing: Always active
│   └── Controls: Autocomplete, import resolution, errors
│
├── TERMINAL ENVIRONMENT (settings.json) 
│   ├── Needs: Python path for commands
│   ├── Timing: When terminal opens
│   └── Controls: `python script.py`, `pip install`
│
└── DEBUG ENVIRONMENT (launch.json)
    ├── Needs: Python path + debug config
    ├── Timing: Only during F5 debugging  
    └── Controls: Breakpoints, step-through, variables
